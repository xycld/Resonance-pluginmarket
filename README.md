# Resonance Plugin Market

EZsinging 官方插件商店。

## 浏览插件

访问 EZsinging 应用内「设置 → 插件市场」浏览和安装插件。

## 提交插件

参见 [CONTRIBUTING.md](./CONTRIBUTING.md)。

## 仓库结构

| 路径 | 说明 |
|------|------|
| `plugins-list/` | 插件注册信息（PR 提交） |
| `plugins-data/` | 打包好的插件 ZIP（CI 自动生成） |
| `plugins.json` | 插件索引（CI 自动生成） |
| `scripts/` | CI 自动化脚本 |
